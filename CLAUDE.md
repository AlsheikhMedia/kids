## Project

**الشيخ ميديا — ركن الأطفال** (Alsheikh Media Kids Corner) — free educational games for Arab children under 5. Arabic RTL first, no English in game UIs. Live at kids.alsheikhmedia.com.

## Commands

```bash
npm run dev          # Dev server with HMR
npm run build        # Vite build + auto-generate CSP script hashes
npm run check        # Type-check all Svelte/TS files (svelte-check)
npx wrangler pages deploy build --project-name kids-corner --branch main  # Deploy to Cloudflare Pages
git pushall          # Push to both GitHub and Codeberg
```

Build runs `scripts/update-csp-hashes.py` post-build — extracts SHA-256 hashes from all inline scripts in built HTML and updates `build/_headers` CSP. This eliminates `'unsafe-inline'` from `script-src`.

## Architecture

**Static SvelteKit** — `adapter-static` with `prerender = true` on the root layout. Zero runtime dependencies. Pure client-side, no backend, no API calls, no auth.

### State Management
`DrawingState` class in `src/lib/stores/drawing.svelte.ts` — Svelte 5 `$state` runes, exported as a module-level singleton. All painting components read/write this directly (no context, no prop drilling).

### Canvas Drawing
Double-buffered: offscreen canvas holds committed strokes, visible canvas composites buffer + in-progress stroke. All coordinates normalized 0-1 (survive resize/rotation). Bezier smoothing via `quadraticCurveTo` midpoint interpolation. DPR-aware sizing.

### Game Registry
`src/lib/games.ts` — `GAMES` array is the single source of truth for all games. Landing page cards, JSON-LD structured data, and routing all derive from this array. Add new games here first.

### Routes
- `/` — Landing page (Hero + game cards + features + about)
- `/paint` — Painting app (canvas + toolbar + fullscreen + parent gate)
- `/games/{colors,numbers,alphabet,puzzle}` — Coming soon placeholders
- `/games/coming-soon` — Generic coming soon page

### Parent Gate
Fullscreen exit button triggers a math problem overlay (single-digit addition in Eastern Arabic numerals). No lock icon, no toggle — just: click exit → solve math → exits fullscreen. ESC key = browser default (fine for tablets, the target device).

## Conventions

- **Arabic RTL** — `<html lang="ar" dir="rtl">`, all layout respects RTL flow
- **Baloo Bhaijaan 2** — self-hosted in `static/fonts/`, variable weight 400-800, Arabic + Latin subsets
- **Touch targets** — minimum 48px (`--touch-target-min`), 56px for primary CTAs (`--touch-target-primary`)
- **No emoji as icons** — use SVG or OpenMoji (self-hosted in `static/icons/`, CC BY-SA 4.0)
- **Scoped CSS** — design tokens in `src/app.css` as CSS custom properties, component styles scoped
- **TypeScript strict** — `interface` over `type` for object shapes, no `any`
- **One component per file**, functional patterns, early returns
- **`prefers-reduced-motion`** — global rule in app.css disables all animations/transitions
- **`:focus-visible`** — global 3px blue outline for keyboard navigation

## Design Tokens (app.css)

Colors: `--color-{red,orange,yellow,green,blue,purple,pink,brown,black,white}`
Surfaces: `--color-canvas-bg` (#fffdf9), `--color-app-bg` (#f8f4f0), `--color-toolbar-bg`
Transitions: `--transition-fast` (120ms), `--transition-normal` (200ms)

## Security Headers

`static/_headers` — Cloudflare Pages format. CSP script hashes auto-generated by build. Also sets HSTS (preload), X-Frame-Options DENY, Permissions-Policy (blocks camera/mic/geolocation/payment).

## Deployment

Cloudflare Pages project `kids-corner`. Custom domain `kids.alsheikhmedia.com` (CNAME to pages.dev). Bot Fight Mode disabled, security level low, HSTS enabled at zone level.
